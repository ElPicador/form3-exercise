# Form3 exercise

## Requirements
* golang v1.11
* [dep](https://github.com/golang/dep)
* make

### Optional
* [pandoc](https://pandoc.org/), for macOS: `brew install pandoc`
* mactex for macOS: `brew cask install mactex`
     

## Run the API

    $ make run

## Run the tests

    $ make test

## (Optional) Generate doc
    
    $ make doc
