# Form3 exercise

## Requirements
* golang v1.11
* [dep](https://github.com/golang/dep)
* make

## Run the API

    $ make run

## Run the tests

    $ make test
    
## Travis build

CI available [here](https://travis-ci.org/ElPicador/form3-exercise)

## (Optional) Generate doc
    
Use a tool to transform Markdown to PDF, like http://markdown2pdf.com.
